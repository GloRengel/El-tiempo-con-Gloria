Descripción del proyecto realizado

El objetivo principal de este trabajo es obtener los conocimiento y el desarrollo de las líbrerias Bootstrap, también el buen manejo de HTML y CSS junto con JavaScript para poder desarrollar una aplicación web responsive y competente.

Objetivo del proyecto

Las APIs utilizadas en este proyecto son:

  • Weatherapi: Para obtener el clima en tiempo real
  • GNews: Para obtener las noticias de España

Las tecnologías utilizadas para este desarrollo son:

  • HTML
  • CSS
  • JavaScript
  • Geminis
  • ChatGPT
  • FreeCodeCamp
